# customer_shopping-_analysis

## 📌 Project Overview
This project analyzes customer shopping behaviour using SQL and Python Power BI to extract business insights from transactional data.

## 🛠 Tools Used
- SQL
- Python (Pandas)
- Power BI

## 📊 Analysis Performed
- Revenue comparison by gender
- Discount impact analysis
- Top 5 highest rated products
- Shipping type purchase comparison
- Subscription spending comparison
- Customer segmentation (New, Returning, Loyal)
- Revenue contribution by age group

## 📂 Project Files
- customer_shopping_behavior.csv (Dataset)
- python code.py (Data analysis)
- SQL Queries PDF
- Dashboard Screenshot
